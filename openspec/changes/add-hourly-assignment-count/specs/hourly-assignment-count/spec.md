## ADDED Requirements

### Requirement: 各時間帯のアサイン数をシフト表に集計行として表示できる
システムは、シフト表のスタッフ行の直前（tbody先頭）に時間帯（午前・午後・夕方）ごとのアサイン数集計行を表示しなければならない（SHALL）。集計行は時間帯ごとに1行ずつ（計3行）、各日付列に「アサイン数/必要人数」の形式で表示する。集計対象は通常スタッフのアサインのみとし、ヘルプスタッフのアサインは含めない。必要人数の取得機能が利用できない場合はアサイン数のみを表示する（例: `2/-`）。集計行はヘルプスタッフセクションより上位（通常スタッフ行の直前）に配置される。

#### Scenario: 集計行が時間帯ごとに表示される
- **WHEN** ユーザーがシフト表画面を開く
- **THEN** スタッフ行の上に「午前」「午後」「夕方」の3行の集計行が表示される

#### Scenario: 各日・各時間帯のアサイン数と必要人数が表示される
- **GIVEN** ある日の午前に通常スタッフ3名がアサインされており、必要人数が6である
- **WHEN** シフト表を表示する
- **THEN** その日の午前集計行のセルに「3/6」と表示される

#### Scenario: アサイン数が必要人数を下回る場合は警告色で表示される
- **GIVEN** ある日の午前のアサイン数が3であり、必要人数が6である（アサイン数が必要人数を下回る）
- **WHEN** シフト表を表示する
- **THEN** その日の午前集計行のセルがオレンジ系の警告色で表示される

#### Scenario: アサイン数が0の場合も警告色で表示される
- **GIVEN** ある日の午前にアサインが1件もなく、必要人数が1以上である
- **WHEN** シフト表を表示する
- **THEN** その日の午前集計行のセルがオレンジ系の警告色で表示される

#### Scenario: アサイン数が必要人数以上の場合は通常色で表示される
- **GIVEN** ある日の午前のアサイン数が6であり、必要人数が6である（ちょうど充足）
- **WHEN** シフト表を表示する
- **THEN** その日の午前集計行のセルが通常のグレー系の色で表示される（警告色ではない）

#### Scenario: 必要人数が0の場合は通常色で表示される
- **GIVEN** ある祝日の午前の必要人数が曜日別デフォルト値（0）として自動設定されており、アサイン数も0である
- **WHEN** シフト表を表示する
- **THEN** その日の午前集計行のセルに「0/0」が通常色で表示される

#### Scenario: ヘルプスタッフのアサインは集計行に含まれない
- **GIVEN** ある日の午前に通常スタッフ2名・ヘルプスタッフ1名がアサインされており、必要人数が6である
- **WHEN** シフト表を表示する
- **THEN** その日の午前集計行のセルに「2/6」と表示される（ヘルプスタッフ分は含まない）

#### Scenario: 必要人数の取得機能が未設定の場合はアサイン数のみを表示する
- **GIVEN** 必要人数の取得機能が渡されていない状態でシフト表を表示する
- **AND** ある日の午前に通常スタッフ2名がアサインされている
- **WHEN** シフト表を表示する
- **THEN** その日の午前集計行のセルに「2/-」と表示される（必要人数が未設定であることを示す）
