## ADDED Requirements

### Requirement: スタッフ一覧をドラッグ&ドロップで並び替えられる

システムは、スタッフ管理画面においてスタッフ一覧の並び順をユーザーが自由に変更できなければならない（SHALL）。並び替えは Pointer Events API を通じて、`pointerType` に基づいて操作方法を切り替える。

- **マウス操作（`pointerType: 'mouse'`）**: リストアイテム左端のドラッグハンドルをマウスでドラッグし、目的の位置でドロップすることで並び替えを行う
- **タッチ操作（`pointerType: 'touch'`）**: ドラッグハンドルを約 300ms 長押しした後、指を動かすことで並び替えを行う
- **スコープ外**: キーボードによる並び替えはサポートしない
- **スコープ外**: ドラッグ中の自動スクロールはサポートしない

並び替えは挿入型（`fromIndex` のアイテムを配列から取り出し `toIndex` の位置に挿入）で実装する。並び替え確定（ドロップ確定時＝`pointerup` イベント発生時）に LocalStorage へ保存する。

#### Scenario: PC でドラッグ&ドロップしてスタッフを並び替えられる

- **GIVEN** スタッフが複数件登録されている
- **WHEN** ユーザーがドラッグハンドルをマウスでドラッグし、別のアイテム位置でドロップする
- **THEN** ドラッグしたスタッフがドロップ先の位置に移動し、残りのスタッフが詰め直されて表示される
- **AND** 変更後の順序がドロップ確定時（`pointerup`）に LocalStorage に保存される

#### Scenario: ドラッグ中のアイテムは視覚的にフィードバックが表示される

- **GIVEN** スタッフが複数件登録されており、ドラッグ操作が開始されている
- **WHEN** ユーザーがドラッグハンドルをドラッグし、別のアイテムの上にカーソルを移動させる
- **THEN** ドラッグ中のアイテムは半透明（opacity-50）で表示される
- **AND** ドラッグ中にカーソル/タッチが重なっているアイテムにボーダーハイライトが表示される

#### Scenario: モバイルで長押し後にドラッグしてスタッフを並び替えられる

- **GIVEN** スタッフが複数件登録されている
- **WHEN** ユーザーがドラッグハンドルを 300ms 以上長押しした後、指を別のアイテム位置まで動かして離す
- **THEN** ドラッグしたスタッフがドロップ先の位置に移動し、残りのスタッフが詰め直されて表示される
- **AND** 変更後の順序が LocalStorage に保存される

#### Scenario: モバイルで長押し前に指を離してもドラッグが開始されない

- **GIVEN** スタッフが複数件登録されている
- **WHEN** ユーザーがドラッグハンドルを 300ms 未満でタップして離す
- **THEN** ドラッグ操作は開始されず、スタッフ順序は変化しない

#### Scenario: ドラッグをキャンセルしても順序が変わらない

- **GIVEN** スタッフが複数件登録されており、ドラッグ操作が開始されている
- **WHEN** ドラッグ中にポインターキャンセルイベント（`pointercancel`）が発生する
- **THEN** スタッフ順序は変化しない

#### Scenario: 同じ位置にドロップしても順序が変わらない

- **GIVEN** スタッフが複数件登録されており、ドラッグ操作が開始されている
- **WHEN** ユーザーがドラッグを開始した位置と同じ位置でドロップする
- **THEN** スタッフ順序は変化しない

#### Scenario: スタッフが1件のみの場合はドラッグ操作が無効となる

- **GIVEN** スタッフが1件のみ登録されている
- **WHEN** ユーザーがドラッグハンドルを `pointerdown` し、`pointermove` を発生させる
- **THEN** ドラッグ操作は開始されず、`draggingIndex` は null のまま変化しない
- **AND** ドラッグハンドルは視覚的に表示される

#### Scenario: 各リストアイテムにドラッグハンドルが表示される

- **GIVEN** スタッフが1件以上登録されている
- **WHEN** スタッフ管理画面を表示する
- **THEN** 各スタッフのリストアイテム左端にドラッグハンドル（グリップアイコン）が表示される
- **AND** ドラッグハンドルにカーソルを合わせると `cursor-grab` スタイルが適用される

## MODIFIED Requirements

### Requirement: スタッフ一覧を表示できる（変更）

既存要件に以下のシナリオを追加する。

#### Scenario: 並び替え後の順序でスタッフ一覧が表示される

- **GIVEN** スタッフの並び替えが行われ、その順序が LocalStorage に保存されている
- **WHEN** ユーザーがスタッフ管理画面を開く（または再読み込みする）
- **THEN** 並び替え後の順序でスタッフ一覧が表示される

> **連動仕様**: スタッフ一覧の並び順は、シフト表（shift-schedule-view）のスタッフ行の表示順序と連動する。スタッフ一覧を並び替えると、シフト表のスタッフ行も同じ順序で表示される。詳細は `specs/shift-schedule-view/spec.md` を参照。
